# dotfiles (alpha)

![Screenshot](https://github.com/jlopezcur/dotfiles/raw/master/screenshot.png)

## Install from a clean Debian 10 (Buster)

```shell
sudo apt install git
git clone git@github.com:jlopezcur/dotfiles.git
cd dotfiles
./install.sh
```
