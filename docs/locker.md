# Locker

I use the [swaylock](https://github.com/swaywm/swaylock) for the cool indicator
:) but with a custom script for capture the screen and blur as the lock
background.

Also use:

- grim
- imagemagick

All the needed packages:

```sh
yay -S imagemagick swaylock grim
```
