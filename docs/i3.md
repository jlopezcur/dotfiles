# [i3wm](https://i3wm.org/)

## Tearing

i3 does not properly implement double buffering hence tearing or flickering may
occur. For this task we use Compton.

- https://github.com/i3/i3/issues/661

## i3 Customs

### Programs

- `$mod+d`: rofi
- `$mod+p`: gpick
- `$mod+Shift+x`: i3lock
- `$mod+z`: scrot -s
- `$mod+c`: qalc
