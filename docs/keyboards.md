# Keyboards

## Keychron K6

https://www.keychron.com/products/keychron-k6-wireless-mechanical-keyboard

For F[n] keys:

```sh
# /etc/modprobe.d/hid_apple.conf

options hid_apple fnmode=2 swap_opt_cmd=1
```
